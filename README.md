# Stockyard Saddlebag

**File sharing.** Upload, share a download link, set expiry or max downloads. Like WeTransfer but self-hosted. Single binary, no external dependencies.

## Free vs Pro

| Feature | Free | Pro ($2.99/mo) |
|---------|------|----------------|
| Files | 10 | Unlimited |
| Max file size | 50MB | Unlimited |
| Password protection | — | ✓ |
| Expiry links | ✓ | ✓ |
